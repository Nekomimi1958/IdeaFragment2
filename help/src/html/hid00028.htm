データファイルの仕様

IdeaFragment2 のデータファイルは、.IFD という拡張子のテキストファイル(UTF-8)です。
テキストですので、直接編集したり、他の用途に用いることも可能ですが、データ書式が崩れると正しく読み込めなくなる恐れがありますので注意してください。

データ書式の概略は以下の通りです。

[Fragment] セクション

>識別子
キー=値
…
…
<
という書式で１個の断片を表します。

識別子
>TXT:通常断片
>PLT:表札
>SYM:標識
>IMG:画像
>JNC:関係線の分割点

キー
ID:内部的に用いられる永続的な固定識別番号
TX:テキスト内容
GP:所属グループ番号の並び
XP:X位置
YP:Y位置
WD:幅(画像の場合は倍率の分子)
HI:高(画像の場合は倍率の分母)
FG:文字色
BG:背景色
LK:リンク先
CT:作成日時
MT:更新日時

[Connection] セクション

>識別子
キー=値
…
…
<
という書式で１本の関係線を表します。

識別子
>GCL:グループ関係線
>FCL:断片関係線
>FGL:断片からグループへの関係線
>GFL:グループから断片への関係線

キー
FR:始点(グループ番号または断片ID)
TO:終点(グループ番号または断片ID)
LT:線タイプ
LW:線幅
AT:矢の形
AS:矢のサイズ
TX:ラベル内容
OP: 1: ラベルを反対側に表示

[Group] セクション

>GRP
キー=値
…
<
という書式で１つのグループを表します。

キー
ID:グループの識別番号
VS:表示状態
FC:グループ枠の色
LW:グループ枠の線幅

[Option] セクション

DataTitle:タイトル
ScrWidth:画面の幅
ScrHeigh:画面の高さ
ScrDivMode:画面分割のタイプ
ScrDivX:画面分割線の横分割数または分割幅等
ScrDivY:画面分割線の縦割数または分割幅
ScrDivLw:画面分割線の線幅
ColBackground:画面背景色(個別)
ColDivLine:画面分割線の色(個別)
ColGrpFrame:グループ枠の色(個別)
ColGrpLine:グループ関係線の色(個別)
ColFrgLine:断片関係線の色(個別)
ColLinLbl:関係線ラベルの文字色(個別)
ColTitle:タイトルの文字色
FontFrg:断片のフォント(個別)
FontNam:表札のフォント(個別)
FontSym:標識のフォント(個別)
