オプション設定 - 断片

断片デフォルトサイズ
断片を作成する際のデフォルトのサイズを桁数・行数で指定します。
表札デフォルトサイズ
表札を作成する際のデフォルトのサイズを桁数・行数で指定します。
画像デフォルト倍率
画像断片を作成する際のデフォルトの表示倍率を指定します。
1／1で原寸、1／2で50％、2／1では倍寸というようになります。
改行
テキストの改行を行う
チェックすると、プロパティ・ダイアログで断片のテキスト入力時に改行を入れることができます。この時、 改行はReturnキーで がチェックされているとReturnキーで、そうでない場合は Ctrl+Returnキーで改行します。
改行文字列
ファイル保存やクリップボードへのコピーの際に用いられる変換文字列です。テキスト中では使用しないような文字列を指定してください(デフォルトでは ＼)。
※保存時に、テキスト中の改行コードは指定した改行文字列に変換されます。編集欄でこの文字列を使うと改行になってしまうので注意してください。

なお、 テキストの改行を行う および 改行はReturnキーで がチェックされていて、さらに Ctrl+ReturnキーでOK をチェックすると、テキスト内容入力欄で Ctrl+Returnキーを押した時に OK ボタンが選択されます。

新規作成位置
新規作成時の断片の出現位置を指定します。
片付け位置
編集メニュー -「片付け」の際の位置を指定します。
移動時カーソル
断片を移動する時のマウスカーソルの形を指定できます。
新規作成時に無所属
新規作成時に断片を無所属とします。ただしグループ枠内のダブルクリックで作成した場合は、そのグループになります。
チェックされていない場合には、以下の優先順位にしたがってグループが決まります。
  1. あるグループの断片だけが選択されていれば、そのグループ
  2. グループリストで選択されているグループ
  3. グループリストが非表示なら、表示状態にある最初のグループ
  4. それもなければ無所属
自動的に行数増
断片テキストがデフォルトの行数を超えた時、自動的に行数が増やされます。
この設定は、ファイルメニュー - インポート - 「テキスト取り込み」や、編集メニュー - 「クリップボードから新規」を実行する際にも適用されます。
テキスト入力の右端で折り返す
プロパティ(新規作成)・ダイアログの断片テキスト編集欄で、表示を右端で折り返します。
テキスト入力時IMEオン
プロパティ(新規作成)・ダイアログの断片テキスト編集欄でIMEが自動的にオンになります。
なお関係線のラベル入力時にもこの設定が反映されます。
リンクをアイコンで表示
リンクマークの代わりに、リンク先ファイルに関連付けられているアイコン(16×16)を表示します。
リンク先を相対パスで表す
リンク先をリンク元からの相対パスで表示し、またデータファイルにも相対パスで保存します。相対パスにしておくと、リンク間のディレクトリ構造を保ったまま別の場所に移動させても、リンクが切れません。
なおリンク先とリンク元のドライブが異なる場合は、絶対パスになります。
画像の縮小・拡大アルゴリズム
画像断片を縮小・拡大表示する際に用いる画像補間法を選択します。
なおメタファイルの場合は、この設定にかかわらず、ベクタイメージとして処理されます。
